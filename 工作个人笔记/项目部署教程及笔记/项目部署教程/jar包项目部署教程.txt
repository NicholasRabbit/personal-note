SpringBoot一般以jar包的形式部署，一个jar包相当于一个小项目，里面内嵌了tomcat
1,把SpringBoot打成的jar包上传到指定目录，例如/opt/package
  (1)在该目录下执行,以下方式shell窗口关闭项目也停止
  java -jar firstApplication.jar > tempLog.txt:生成日志到当前目录下，tempLog.txt自动创建， 此命令默认使用8080端口，不建议使用
  java -jar firstApplication.jar  --server.port=9090 > tempLog.txt: 自定义端口，注意要在Linux防火墙设置9090例外，否则无法访问
  一般在SpringBoot的项目的相关配置文件中自定义端口，在application.yml等文件中
  (2)后台继续运行项目写法：
  nohup java -jar firstApplication.jar  --server.port=9090 > tempLog.txt : 在后台运行
  或 ： nohup java -jar coal_jt808Serve.jar &
        tail -f nohup.out 
  防火墙设置自定义的9090例外，否则外部无法访问
   firewall-cmd --zone=public --add-port=9090/tcp --permanent
   firewall-cmd --reload    #重启firewall
