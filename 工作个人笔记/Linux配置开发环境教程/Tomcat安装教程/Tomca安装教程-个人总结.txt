这里以tomcat8为例
安装步骤：
1，官网下载tomcat的压缩包，解压到/usr/local/tomcat8
   tar  -zxvf  apache-tomcat-8.5.72.tar.gz  -C  /usr/local/tomcat8
2，测试到解压缩包的bin目录下
   cd  /usr/local/tomcat8/apache-tomcat-8.5.72/bin
   ./startup.sh : 启动
   ./shutdown.sh : 停止
3，配置CATALINA_HOME
   vi /etc/profile
   添加以下内容：
   CATALINA_HOME=/usr/local/tomcat8/apache-tomcat-8.5.72
   export JAVA_HOME PATH CLASSPATH CATALINA_HOME
   ~~-#>source /etc/profile : 重新加载配置
4，配置tomcat的快捷启动方式
   cd /etc/rc.d/init.d/  : 切换到该目录
   touch tomcat ： 新建文件tomcat
   chmod +x tomcat " 添加执行权限
   添加以下内容：
        #!/bin/bash          
	#chkconfig:- 20 90
	#description:tomcat
	#processname:tomcat
	TOMCAT_HOME=/usr/local/tomcat/apache-tomcat-8.5.55    #此路径按照个人电脑的路径
	case $1 in
	 start) su root $TOMCAT_HOME/bin/startup.sh;;
	 stop) su root $TOMCAT_HOME/bin/shutdown.sh;;
	 *) echo "require start|stop" ;;
	esac
   配置JAVA_HOME和JRE_HOME,否则报错找不到jdk，（程序羊的步骤出错解决办法）
   在tomcat的bin目录下，编辑catalina.sh文件，加入
	   #   JAVA_HOME       Must point at your Java Development Kit installation.
	   #                   Required to run the with the "debug" argument.
	   export JAVA_HOME=/usr/local/jdk8/jdk1.8.0_121      
	   #
	   #   JRE_HOME        Must point at your Java Runtime installation.
	   #                   Defaults to JAVA_HOME if empty. If JRE_HOME and JAVA_HOME
	   #                   are both set, JRE_HOME is used.
	   export JRE_HOME=/usr/local/jdk8/jdk1.8.0_121/jre
   以上配置完之后执行以下命令可开启tomcat,不用去tomcat的bin目录了
   service tomcat start ：开启
   service tomcat stop  ：关闭
	
5，设置tomcat开机自启动
   cd /etc/rc.d/init.d/  : 切换到该目录
   chkconfig --add tomcat  ：这里“tomcat”就是上面的快捷方式名
   chkconfig tomcat on
   chkconfig --list tomcat :检查是否开机自启动设置成功2345on表示成功  0:off   1:off   2:on    3:on    4:on    5:on    6:off 

6，防火墙设置Tomcat例外，否则外部无法访问
   firewall-cmd --zone=public --add-port=8080/tcp --permanent
   firewall-cmd --reload #重启firewall
   
   关闭例外的端口，设置完毕要重启
   firewall-cmd --zone=public --remove-port=8080/tcp --permanent
   firewall-cmd --reload #重启firewall

如果系统没有防火墙
yum install firewalld
systemctl unmask firewalld
systemctl enable firewalld
systemctl start firewalld
yum install iptables-services
然后开启防火墙
systemctl  start firewalld.service ： 停止防火墙
systemctl  stop firewalld.service ：开启防火墙
systemctl  enable firewalld.service ： 防火墙开机启动
systemctl disable firewalld.service ： 禁止防火墙开机启动

其它常用命令
firewall-cmd --list-ports : 显示防火墙允许的所有端口
firewall-cmd --state : 查看防火墙是否开启
netstat -tlnp|grep 8080 ： 查看占用8080端口的占用情况

7,Linux下查看Tomcat服务器控制台输出
  tail  -f  /tomcat/.../logs/catalina.out : 可动态查看后台的打印输出
