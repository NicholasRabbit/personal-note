#### 2，常用命令

```shell
1, :set fileencoding ：vim查看编码格式

2, putty设置vim编辑器使用数字小键盘方法
   Terminal--> Features --> Disable application keypad mode，勾选此项

3, vim中批量替换命令
   :%s/abc/xyz  把文件中所有的abc替换为xyz

4, Ctrl + o : 光标回退到上一个位置
   Ctrl + i : 光标前进到下一个位置
   
5, :f或ctrl+g -- 显示文档名，是否修改，和光标位置

6, dt) : 删除到下一个右括号前的所有内容。
```
